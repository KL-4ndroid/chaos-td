# ADR-005｜Phaser 4 Baseline

- Status：Accepted with Validation Gate
- Date：2026-07-20

## Context

綠地專案；官方提供 Phaser 4.2.1，但舊範例多基於 Phaser 3。

## Decision

- M0 預設 Phaser 4.2.1。
- 驗證 Vite、TypeScript、Scale、Scene、Input、Asset。
- 實際阻塞時需新 ADR 才可降 Phaser 3.90.x。

## Consequences

採官方主線，但 Cursor 不可照抄 Phaser 3 API。
