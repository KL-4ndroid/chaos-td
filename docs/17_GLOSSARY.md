# 17｜Glossary

| Term | Definition |
|---|---|
| AI | deterministic 規則型對手 |
| Armor | 降低 Shield 後的 HP Damage |
| Bounty | 擊殺來襲怪物取得的 Gold |
| Command | 玩家／AI 對 Core 的意圖 |
| Config Version | 完整規則與數值版本 |
| Determinism | 相同輸入得到相同結果 |
| Domain Event | Core 已發生的規則事實 |
| Fixed-point | 整數表示小數，如 milli-tile |
| Hitscan | 攻擊 Tick 立即結算 |
| Income | 每 10 秒 Gold，送怪永久增加 |
| Lane | 一方防守的路線 |
| Leak | 怪物到終點扣 HP |
| Local Perspective | p1／p2 對本機玩家的解讀 |
| Net Worth | Gold＋塔出售價 |
| Permille | 千分比，1000‰ = 100% |
| Replay | Seed＋Config＋Accepted Commands |
| RESOLVING | 停止新 Command，清既有 Queue／怪物 |
| Seed | PRNG 初始輸入 |
| Send Queue | 已付款等待送出的怪物 |
| State Hash | Canonical State 摘要 |
| TPS | Simulation Ticks Per Second |
